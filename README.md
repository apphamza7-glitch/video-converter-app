# Video Converter App

A simple starter project with:

- Node.js + Express backend
- React frontend
- API endpoint for conversion requests

## Structure

- backend/ contains the Express server
- frontend/ contains the React app

## Run locally

1. Install backend dependencies:
   ```bash
   cd backend
   npm install
   npm start
   ```

2. In another terminal, install frontend dependencies:
   ```bash
   cd frontend
   npm install
   npm start
   ```

3. Open http://localhost:3000
